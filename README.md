# دیتابیس SQLite3 بسیار غنی و کاربردی مخصوص تعرفه‌های گمرکی HS Code با 10000 رکورد

## معرفی
این مخزن شامل یک دیتابیس **SQLite3** بسیار کامل و ساختارمند است که برای کار با **تعرفه‌های گمرکی (HS Code)** طراحی شده است. این دیتابیس می‌تواند به‌عنوان یک منبع داده قابل‌اعتماد برای توسعه‌دهندگان وب، اپلیکیشن‌های موبایل، سرویس‌های گمرکی و ابزارهای تحلیل تجارت بین‌الملل مورد استفاده قرار گیرد.

---

## ساختار دیتابیس
این دیتابیس شامل **۳ ستون اصلی** است:

| ستون | توضیح |
|------|--------|
| `hs_code` | کد تعرفه گمرکی (کلید اصلی) |
| `description_fa` | توضیحات تعرفه به زبان فارسی |
| `description_en` | توضیحات تعرفه به زبان انگلیسی |

ساختار نسخه کوتاه و نسخه کامل کاملاً یکسان است و هیچ‌گونه رمزگذاری یا رکورد تبلیغاتی در نسخه اصلی وجود ندارد.

این دیتابیس برای توسعه‌دهندگان و شرکت‌هایی که قصد ساخت ابزارهای زیر را دارند بسیار مناسب است:
